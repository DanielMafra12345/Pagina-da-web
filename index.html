Projeto de Página na Web

Esse projeto será uma forma de desenvolver a linguagem de programação HTML, CSS e JAVA SCRIPT

Linguagens

Nesse site nós vamos descrever um pouquinho de cada uma das seguintes linguagens:

  • Blocos: Linguagem visual baseada em blocos, usada para criar programas arrastando e encaixando peças, ideal para iniciantes.
  • Python: Linguagem simples e poderosa, com sintaxe fácil. Usada em automação, ciência de dados, inteligência artificial e desenvolvimento web.
  • JavaScript: Linguagem essencial para criar interatividade em sites, funcionando diretamente no navegador.
