Projeto de Página na Web

Esse projeto será uma forma de desenvolver a linguagem de programação HTML, CSS e JAVA SCRIPT

Linguagens

Nesse site nós vamos descrever um pouquinho de cada uma das seguintes linguagens:

  • Blocos: Linguagem visual baseada em blocos, muito usada em plataformas como Scratch. Permite criar programas arrastando e encaixando blocos, o que facilita o aprendizado para iniciantes e crianças.
  • Python: Linguagem simples e poderosa, conhecida por sua sintaxe fácil de entender. É muito usada em automação, desenvolvimento web, ciência de dados, inteligência artificial e muito mais.
  • JavaScript: Linguagem usada principalmente para criar interatividade em sites. Funciona direto no navegador e é essencial para o desenvolvimento web moderno.
